# 航空工具箱微信小程序功能说明

## 1. 产品概述

航空工具箱是一款面向航空出行场景的信息查询小程序，提供航空公司、机场、城市时间和货币换算等常用工具。

当前页面采用统一的浅色航空主题界面，顶部标题区域会自动适配手机状态栏和微信胶囊按钮，避免标题与手机时间、信号、电量区域重叠。

## 2. 首页

首页展示四个主要功能入口：

- 航空公司：查询航空公司代码和客服电话。
- 机场信息：查询机场三字码和客服电话。
- 城市时间：查询常用航空城市的当前时间。
- 汇率换算：进行常用币种之间的金额换算。

首页同时展示航空工具箱名称、功能简介和数据更新提示。

## 3. 航空公司信息查询

页面支持按照以下内容进行搜索：

- 航空公司中文名称。
- 航空公司英文名称。
- 航空公司两字码或其他代码。
- 航空公司客服电话。

查询结果以信息卡片展示，包含：

- 航空公司名称和英文名称。
- 航空公司标识和两字码。
- 客服热线。
- “立即呼叫”按钮。

页面提供分类筛选：

- 全部。
- 国内航司。
- 国际/地区。
- 廉价航空。

搜索框左侧有返回首页按钮，底部导航中的“工具箱”和“关于”均可正常使用。

## 4. 机场信息查询

页面支持按照以下内容进行搜索：

- 机场名称。
- 机场三字码。
- 所在城市。
- 所在省份。
- 机场客服电话。

查询结果以信息卡片展示，包含：

- 机场中文名称和英文名称。
- 机场三字码。
- 所在省份、城市或区域。
- 机场等级。
- 客服电话。
- “立即拨号”按钮。

页面提供分类筛选：

- 全部。
- 大型枢纽。
- 省会机场。
- 支线机场。

“立即拨号”按钮与卡片内容区域保持统一的右侧留白，点击后调用手机拨号功能。

## 5. 城市时间

页面覆盖全球主要航空枢纽城市，并按区域分类展示。当前提供以下区域按钮：

- 全部。
- 亚洲。
- 欧洲。
- 美洲。
- 中东。
- 非洲。
- 大洋洲。

页面展示城市当前时间，并根据城市时区自动计算当地时间、UTC 偏移和与设备本地时间的差异，包含夏令时变化。

页面支持搜索：

- 城市中文名。
- 城市英文名。
- 国家或地区名称。
- UTC 偏移量。
- 城市相关机场代码。

时间信息会持续刷新，卡片展示城市名称、日期、当前时间、UTC 偏移和与设备本地时间的差异。当前共收录 79 个城市，其中亚洲 43 个，覆盖亚洲、欧洲、美洲、中东、非洲和大洋洲的主要航空城市。

搜索框左侧提供返回首页按钮，底部“工具箱”按钮可以返回首页。

## 6. 汇率换算

汇率页面支持选择原币种和目标币种，并输入金额进行换算。

当前页面支持：

- 输入原币种金额，自动计算目标币种金额。
- 直接输入目标币种金额，反向计算原币种金额。
- 切换原币种和目标币种。
- 使用页面底部数字键盘输入数字、小数点和退格。
- 查看常用航空汇率示例。

金额展示和输入统一保留小数点后 2 位，删除到最小值时显示为 0.00，避免出现浮点数计算产生的多余小数位。

页面底部提供可展开和收起的汇率计算器。点击货币兑换卡片会自动展开计算器，点击其他页面区域会收起计算器。计算器不调用手机系统键盘，使用页面内数字键盘输入，并在当前编辑金额末尾显示光标。

上方原币金额和下方目标金额均可独立输入，输入第一个数字时会替换当前值，并自动计算另一侧金额。上下金额框都支持数字、小数点和退格操作，小数点后最多输入 2 位。

页面提供独立的返回首页按钮，货币兑换卡片保持完整宽度显示；底部“工具箱”按钮可以返回首页。

## 8. 统一导航

所有主要页面底部均提供统一导航：

- 工具箱：返回首页。
- 查询：表示查询类工具页面入口。
- 收藏：保留为后续功能入口。
- 关于：进入关于页面。

航司、机场、城市时间和汇率页面的顶部飞机按钮、搜索或主输入区域左侧返回按钮均可返回首页。

## 9. 当前功能边界

当前版本主要提供静态信息查询和示例换算功能，不包含：

- 航班动态查询。
- 机票价格查询。
- 航班延误预测。
- 实时汇率接口。
- 用户登录和账号体系。
- 收藏数据同步。
- 后台管理功能。

页面信息仅供快速查询参考，具体航空公司、机场、时间和汇率信息应以官方渠道为准。
